Buttons

  • :hover pseudo element with transform: translateY to raise button on mouseover
  • box-shadow to create feeling of depth when raised
  • reduce transform: translateY and reduce box-shadow: when pressed
  • ::after element with 100% width & height and lower z-index hidden behind button
  • use transform: scale & opacity: 0 on ::after element when hovered to make it grow then disappear!

Gradient Text

  • Set background-image: linear-gradient(to right, color1, color2); on text element
  • add -webkit-background-clip: text; which will clip the gradient background to match the text element
  • add color: transparent; to make the background gradient shine through!
  • The additional funkiness over yonder was done by adding transform: skewY(1.5deg) skewX(10deg) scale(1.1); on text when hovered
  • Also used text-shadow to again give a feeling of depth

Nav Hover Effect

  • Add default styling under pseudo elements :link, :hover
  • Give default style a background-image: linear-gradient(120deg, transparent 0%, transparent 50%, COLOR 50%);
  • Then to hide the background color, use background-size: 250%; (play around with % to get right size for your particular case)
  • Then on :hover reset to background-position: 100%;
  • Remember to use transition: on default text to make it smooth!

Form: Placeholders

  • add border-bottom: green; for :focus and a red bottom border for :focus:invalid for basic HTML form validation
  • Put your input before your label in HTML and use display: block; on text input to make sure label appears underneath input
  • Use input:placeholder-shown + label as selector and add opacity: 0; visibility: hidden; on label
  • The + sign selects the element immediately after the previous element, so when a placeholder is shown the label will not be visible
  • Also use transform: translateY() to create animation effect when label appears (don't forget transition!)
